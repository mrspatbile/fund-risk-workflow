# AIFM Hedge Fund Output Gallery

This folder contains the tables and plots generated by the AIFM Hedge Fund risk-monitoring notebook.

The outputs are numbered according to the order defined in the notebook, so they can be reviewed directly from this folder.

Source notebook: [`aifm_hedge_fund.ipynb`](../../notebooks/fund_risk_monitoring/aifm_hedge_fund.ipynb)

Note: These outputs are generated from simulated fund, position, and market data. They illustrate the risk-monitoring workflow and are not intended to represent a production report.
